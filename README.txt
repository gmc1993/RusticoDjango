# RusticoProducciones-Proyecto-Grupo24
Este es el Trabajo Práctico Obligatorio de Desarrollo Web HTML, CSS y Javascript (TPO) realizado por el grupo 24 de la comisión 23032, del curso FullStack Python para 
Front End del programa de enseñanza en Programación Codo a Codo (4.0). La temática elegida fue la elaboración de una página web para una pequeña productora independiente
llamada “Rústico Producciones”.  Los miembros integrantes del equipo son:

-	Pers Prato (DNI: 96.099.849)
-	Diego Roman (DNI: 35.322.608)
-	Gonzalo Cacheiro (DNI: 35.996.318).
-	Andrea Villamizar. (Documento Extranjero: 20.302.541)

Esta idea surgió de Gonzalo, uno de los integrantes del grupo, ya que él es allegado de los creadores de la productora, conocidos en la universidad de cine, y al empezar
a ver programación web se le ocurrió como propuesta de proyecto realizarles una web debido a que así el proyecto tendría un carácter más real; más que solo ser una página
usada para práctico, podría ser algo funcional que pueda ser usado en un dominio y así darnos también a los integrantes una experiencia demostrable que tenga funcionalidad.
Los otros miembros estuvimos de acuerdo, con lo que nos pusimos manos a la obra, y este sería nuestro resultado.

El rol cumplido por cada uno de los integrantes fue:
•	Gonzalo Cacheiro:
Se encargó de la sección “Proyectos”, con la premisa de mostrar proyectos desarrollados por la productora, se inspiró en el diseño de netflix y otros servicios de streaming
para mostrarlos de una forma moderna, cómoda y familiar. Haciendo tres carruseles para cada tipo de proyecto con sus respectivos posters. Al realizar click sobre alguno abre
un menú para ver el video en un iframe, junto con la ficha técnica y la descripción. Utilizó iconos de FontAwesome para las flechas de los carruseles y el botón para cerrar
el menú. Realizó el carrusel solo con CSS y JavaScript sin la utilización de algún framework, y lo diseñó para que sea una columna al pasar a navegación por celular.

•	Andrea Villamizar:
Se encargó de la creación e implementación de una API local, utilizada en la sección de “nosotros”, para mostrar al equipo de trabajo de la productora. Utilizando un archivo
Json que contenía toda la información de la API, en conjunto con un framework de JS, enlazó e hizo visible la API de manera dinámica para el cliente. También realizó el
formulario de contacto con una validación con JS de modo que fuese funcional, en la misma página se implementó el uso de iconos de FontAwesone con acceso directo a las redes
sociales del proyecto como un Iframe de ubicación.

•	Pers Prato.
Realizó el header dinámico compartido por todas las páginas de la web, y lo volvió adaptable y responsivo en medida para diferentes puntos de corte mientras cambia el ancho
de la página, de forma que sea compatible con diferentes dispositivos. También fue encargado de la página de inicio, realizando todo el maquetado y diseño de esta, y fue el
responsable de unificar el código de los otros integrantes del grupo, para darles forma y ver que corrieran bien junto con el header y las querys. En esto último también
participó Gonzalo con la sección de proyectos.

Notas: Andrea se encargó de realizar las páginas de Nosotros y de Contacto, Gonzalo la de proyectos, y Pers la de Inicio, y unificó todas para que fueran compatibles y
funcionales entre sí, y Gonzalo realizó toda la página de proyectos. La animación más destacada fue la utilizada por Andrea en la sección de nosotros en la que se dan
vuelta las cards de los integrantes de la productora. Diego Roman dimitió del curso, no contribuyó ni aporto a la realización del proyecto.
